// What it does

The problem with conventional power flow analysis

Traditional bus-wise Newton-Raphson methods tell you if your system is unstable — but not which line is causing it. LineWise solves that.

INPUT

Upload IEEE CDF format data

Upload your distribution network case file in standard IEEE Common Data Format (Excel). Bus data, branch data, and system parameters are parsed and validated automatically.

SOLVER

Line-wise Newton-Raphson engine

Uses squared voltage magnitudes (U = V²) as state variables, improving numerical stability near collapse. Solves without constructing the bus admittance matrix.

VCI

Voltage Collapse Index per line

Each transmission line gets a VCI score extracted directly from the Jacobian. Lines approaching collapse are identified before the system fails — something bus-wise methods cannot do.

VISUAL

Interactive network dashboard

Real-time topology map with colour-coded VCI heatmap, voltage profile charts, phase angle plots, and downloadable results — accessible from any browser, no installation required.

// Voltage Collapse Index

Know which line is failing before it does

The VCI is derived from the Jacobian matrix at convergence. As a line approaches its maximum loadability, VCI approaches zero. Our three-tier threshold model turns this into an actionable signal for engineers and operators.

Status VCI Range Meaning
Stable VCI > 0.3 Line operating normally
Marginal 0.1 – 0.3 Approaching loadability limit
Critical VCI < 0.1 Voltage collapse risk — act now

// Thresholds based on line-wise NR formulation by Mohamed & Venkatesh, IEEE Trans. Power Systems, 2018

// System architecture

Four modules, one workflow

01

Input Module

Parses IEEE CDF Excel files. Validates BUS & BRANCH data. Enforces format rules. Builds solver-ready network model.

02

Solver Unit

Line-wise NR iterations on squared voltages. Assembles Jacobian from line parameters. Converges to 10⁻⁶ tolerance.

03

Output Module

Serializes bus voltages, phase angles, line flows, and VCI to structured CSV/XLSX. Validates before transmission.

04

Visualization

Cytoscape.js topology graph, Plotly.js charts, VCI heatmap. Hosted on Vercel. Download results in one click.
